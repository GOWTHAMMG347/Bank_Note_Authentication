MG Bank Note Authentication System

This project predicts whether a bank note is Authentic or Fake using a Deep Learning model built with TensorFlow/Keras.

Project Details:

  • Variance: Variance of the wavelet transformed image.
  • Skewness: Skewness of the wavelet transformed image.
  • Curtosis: Curtosis of the wavelet transformed image.
  • Entropy: Entropy of the image.

Enter these four parameters from a bank note image to verify its authenticity.

Example Input:
Variance: -6.0598
Skewness: 9.2952
Curtosis: -0.43642
